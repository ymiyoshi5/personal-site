My Projects

Block Paper Scissors (C#)

I worked on this mobile app with my brother using Unity, where we combined the classic mechanics of a Block Blast game with the logic of Rock, Paper, Scissors. My focus was on the back end, ensuring smooth gameplay through key features like validating block placements, highlighting valid moves with color cues (red/green), and implementing algorithms to detect when there are no more moves available—triggering the game over panel.

We are currently working on updates and publishing the app for download!

Troll Escalator Tower (Lua)

This is a Roblox Experience I created during my internship at moze Inc. in Japan. I was in charge of the stage design and implemented the tags to create the escalator effect on the blocks using Lua.

Seeing real reactions from players all over the world to something I built was incredibly exciting—especially watching it grow from 500,000 visits in the first week to over 20 million!
